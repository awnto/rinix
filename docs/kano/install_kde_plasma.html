RNX OS - Install KDE Plasma Desktop on Android - RNX Linux
 
---------

    Updated on : 10/04/2024

	Index - Table


   first install KDE Plasma package 
   then run this command


su


   then run this command


apt update
apt install kde-plasma-desktop




   run this command to set KDE Plasma as default X11 program


echo "dbus-launch startplasma-x11" > /awnto/ux/xstartup 



  Now restart RNX Linux to use KDE Plasma
  it will work on RNX XIO , VNC , XRDP

That's it!
