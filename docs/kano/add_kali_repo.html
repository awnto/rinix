	Index - Table


Step 1 : 

run command

su -c apt edit-sources

   
   then enter your root password
   it open command line text editor
   then add below text at last of file



#### awnto edits
deb http://http.kali.org/kali/ kali-rolling main non-free contrib
#### end awnto edits





then save file
     ctrl+s to save file
   then
     ctrl+x to close editor



Step 2 : 

add pgp key for signature verification

   login to root user

su

   then

wget -q -O - https://archive.kali.org/archive-key.asc | apt-key add




Step 3 : 

all done run as root user to update apt database

apt update




Step 4 :

Enjoy 
All done 
kali repository added to your linux
