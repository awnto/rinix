Solution for libreoffice error in RNX K9

	first install libreoffice
	libreoffice solution for error is to run given commands
	Note : this solution is only for aarch64 == arm64 devices


mkdir -p /prod
wget -O /prod/version https://rinix.awnto.com/rnx-get/files/pla/proc_version

mv /usr/lib/libreoffice/program/oosplash /usr/lib/libreoffice/program/oosplash.ori
wget -O /usr/lib/libreoffice/program/oosplash https://rinix.awnto.com/rnx-get/files/pla/libre_oosplash
chmod 755 /usr/lib/libreoffice/program/oosplash


	all done now open libereoffice



Solution for chromium
	
	first install chromium
	open new terminal in GUI
	run "chromium --no-sandbox" instead of "chromium"
	or you can use firefox

	
Solution for telegram-desktop
	
	first install telegram-desktop
	open new terminal in GUI
	run "telegram-desktop -many" instead of "telegram-desktop"
	or you can use web.telegram.org in firefox

	
Solution for Microsoft Visual Code Studio
	
	first install visual code studio
	open new terminal in GUI
	run "code --no-sandbox"
